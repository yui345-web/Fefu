# Fefu
Решение кейса ВТПШ (NLP анализатор)

Данная модель предназначена для классификации текста по настроению.
Она принимает текст на английском языке и предсказывает одно из 4 настроений:

Радостный 
Грустный
Негативный 
Нейтральный 

Модель обучена на основе LSTM/GRU и использует предобработку текста + эмбеддинги слов.

Для работы модели необходимо установить зависимости:

```pip install tensorflow numpy```

Скачайте модель, и следуйте инструкции в readme файле в ветке telegram_bot
